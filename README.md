# str_header.h
A header only C string library

| Function                                  | Description                                                            |
| :------------                             |   :---:                                                                |
| ```C Str str_new(char *string)```           | Returns a new string                                                   | 
| ```C Str str_concat(Str str1, Str str2)``` | Concatenates 2 strings and returns a new string                        |
